# EcoTrack

![EcoTrack Logo](https://via.placeholder.com/150)

## Overview
EcoTrack is a user-friendly web application designed to help individuals track their daily carbon footprint and receive actionable recommendations to reduce their environmental impact. By integrating with various data sources, EcoTrack provides personalized insights and goals to encourage sustainable living.

## Features
- Real-time carbon footprint tracking based on daily activities
- Personalized tips and goals to reduce environmental impact
- Integration with smart devices and calendars
- Visual analytics to monitor progress over time
- Community challenges to promote eco-friendly habits

## Tech Stack
- Frontend: React.js with TypeScript
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT
- Deployment: Docker, AWS

## Installation
1. Clone the repository:
```bash
git clone https://github.com/miley-bell/EcoTrack.git
```
2. Navigate to the project directory:
```bash
cd EcoTrack
```
3. Install dependencies for backend and frontend:
```bash
cd backend && npm install
cd ../frontend && npm install
```
4. Configure environment variables for backend (e.g., MongoDB URI, JWT secret).
5. Start the backend and frontend servers:
```bash
# In backend folder
npm start

# In frontend folder
npm start
```

## Usage
- Open your browser and go to `http://localhost:3000`
- Register an account or log in
- Connect your smart devices or manually add activities
- View your carbon footprint dashboard and set goals
- Join community challenges and track your progress
